- assets 静态资源
  -  img 图片
  -  .scss 工具函数
- components 组件
- enum 枚举值
- lib 第三方依赖
- pages 页面
  - 文件夹为 一级菜单
  - 二级菜单如果有多个页面 则以文件夹形式
  - 否则则以单文件形式
- plugins 插件和混入对象
- project-config 项目配置
- router 路由
  - 根路径路由 common.js
  - 模块路由 xxx.js
  - 所有路径入口 index.js
- store 结构与router相同
- utils 工具函数



#### 命名规范

1. 文件单词用-隔开
2. 变量用驼峰命名
3. class用-命名也可以使用BEM，id用驼峰命名
4. 避免使用标签选择器 和!important
5. 业务逻辑抽离
